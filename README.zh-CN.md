# 待办

把脑子里的事情变成可执行任务，安排到日历里，再让 Better 帮你持续推进。

[English](./README.md)

## 它解决什么问题

待办适合“一个想法刚刚变成行动”的时刻。你可以用自然语言添加任务，补上日期或提醒，然后在专注列表或日历视图里复盘和安排。

## 你可以用它做什么

- 记录标题、备注、优先级、标签、子任务、截止日期、重复规则和提醒。
- 用日视图、周视图、月视图按时间规划任务。
- 让 Better 查找逾期任务、批量改期、完成 errands 或整理列表。
- 在同一个工作区里看清状态、优先级、提醒和上下文。

## 可以这样问

- “提醒我下周五上午更新护照。”
- “列出逾期待办，把低优先级的移到下周。”
- “把 errands 标签下的事项全部标记完成。”

## 对话里的交互卡片

Todos 已经在 `ui/src/cards/` 加入 React 对话卡片，用来展示 Better 返回的结构化待办结果：

- 待办详情卡展示单个任务的优先级、状态、日程、提醒和标签。
- 待办列表卡聚合多个任务，并显示完成数和待处理数。
- 日程卡把当天计划渲染成按时间排序的紧凑 agenda。
- 批量操作卡总结已完成、已改期、已取消或已更新的任务。

这样对话里的任务结果会更清楚、更可操作，而不是裸 JSON。

## 隐私

待办数据存储在你设备上的 Better 里。它不会访问外部服务，也只使用自己的 connector 存储空间。
